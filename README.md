# Restaurant Management Web App 🍽️

## Overview
A full-stack client-server web application developed to manage restaurant orders, dynamically display menus, and track site statistics. 

## Technical Features
* **Backend Server:** Implemented a robust backend utilizing `Node.js` to route JSON data and serve pages dynamically.
* **Dynamic Templating:** Integrated the `Pug` templating engine to render variable HTML pages and process complex JSON objects for different restaurant menus.
* **Frontend Interface:** Designed interactive, user-friendly frontend interfaces utilizing HTML, CSS, and client-side JavaScript for real-time DOM manipulation.
* **Data Routing:** Managed the flow of structured JSON data between the client and server to keep track of active orders and user sessions.

## Skills Demonstrated
JavaScript, Node.js, HTML/CSS, Pug, JSON, Full-Stack Web Development, Client-Server Architecture.
